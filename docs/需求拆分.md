## US-01 侧边栏文件树

**优先级**: P0 | **工时**: 12h | **依赖**: US-16

**AC**:
- 侧边栏固定 288px 宽度
- 顶部搜索框带 Cmd+K 提示，防抖 300ms
- 文件树可滚动，显示完整目录结构
- 第一层目录显示为大写字母分组标题
- 目录每级缩进 12px，文件额外缩进 20px
- 文件显示时自动附加 .md 扩展名
- 当前文件高亮显示（主题色背景+加粗）
- 右键菜单：目录(新建/重命名/删除)，文件(重命名/删除)
- 底部 "+ 新建" 按钮
- 初始显示空状态提示

---

## US-02 富文本编辑器

**优先级**: P0 | **工时**: 24h | **依赖**: US-04

**AC**:
- 支持 H1/H2/H3 标题
- 支持加粗/斜体/下划线/删除线
- 支持 6 档字体大小 (12px-32px)
- 支持 9 种文字颜色、8 种背景高亮
- 支持行内代码和代码块（行号+语法高亮）
- 支持无序/有序/任务列表
- 支持引用块（6 种边框颜色）
- 支持表格（拖拽调整列宽、右键菜单）
- 支持超链接和水平分隔线
- 符合排版规范：根字号 120%，行高 1.6

## US-03 模式切换

**优先级**: P0 | **工时**: 6h | **依赖**: US-01, US-02

**AC**:
- 默认阅读模式（编辑器只读）
- 阅读模式显示"编辑"按钮（铅笔图标）
- 进入编辑：点击编辑按钮 或 Ctrl/Cmd+E
- 编辑模式显示工具栏和"取消"/"保存"按钮
- 保存按钮无变更时禁用
- 保存中显示加载动画

---

## US-04 Markdown 双向转换

**优先级**: P0 | **工时**: 16h | **依赖**: 无

**AC**:
- 加载时 MD -> HTML
- 保存时 HTML -> MD
- 符合 GFM 规范
- 任务列表正确转换
- HTML -> MD 完整映射：
  - h1/h2/h3 -> #/##/###
  - strong/b -> **text**
  - em/i -> *text*
  - u -> 保留标签
  - s/del -> ~~text~~
  - 行内代码 -> `text`
  - 代码块 -> ```code```
  - 引用 -> > text
  - 链接 -> [text](url)

---

## US-05 全局搜索

**优先级**: P0 | **工时**: 10h | **依赖**: US-01, US-22

**AC**:
- Ctrl/Cmd+K 打开搜索弹窗
- 输入框自动聚焦
- 两种模式：文件名搜索（默认）/ 全文搜索
- 防抖 200ms，最小 2 字符触发
- 显示最近打开文件（最多 5 个）
- 每条结果显示：文件名 + 路径 + 匹配摘要
- 键盘导航：上下移动、Enter 打开、Escape 关闭
- Ctrl/D 切换搜索模式

**优先级**: P1 | **工时**: 6h | **依赖**: US-02, US-03

## 后端 API 故事
