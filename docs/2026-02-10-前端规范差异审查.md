## 二、差异/缺失项

### 1. Biome 配置文件缺失

- **文档要求**：`biome.json` 配置文件
- **现状**：无 `biome.json`，Biome 使用默认行为
- **影响**：中 — 无法自定义 lint/format 规则

### 2. Husky / Git 钩子未设置

- **文档要求**：Husky 9.x，提交前自动检查（类型/格式/测试）
- **现状**：无 `.husky/` 目录，无 commit hooks
- **影响**：低 — 项目尚未初始化 Git 仓库

### 3. React Router 已安装但未使用

- **文档要求**：Data API（Loader/Action），嵌套路由，代码分割
- **现状**：`react-router ^7.13.0` 在 `package.json` 中，但无 `routes.tsx`，无路由配置，`App.tsx` 直接渲染 `EditorView`
- **影响**：中 — 当前单页面场景可以不用路由，但如后续扩展（设置页、多视图）会需要

### 4. Shadcn UI 未正式引入

- **文档要求**：Shadcn UI 组件库，`shared/ui/` 保持原始 Shadcn 组件，Radix UI 无障碍基础
- **现状**：`shared/ui/` 下只有手写的 `Dialog.tsx` 和 `ContextMenu.tsx`，非 Shadcn 标准组件，无 Radix UI 依赖
- **影响**：高 — UI 规范一致性、无障碍支持

### 5. React Hook Form 未安装

- **文档要求**：React Hook Form 7.x + Zod resolver
- **现状**：未安装，表单使用手写 `useState` 控制
- **影响**：低 — 当前表单简单（仅创建/重命名对话框）

### 6. FSD 中间层缺失

- **文档要求**：完整 FSD 分层 `app/` → `views/` → `widgets/` → `entities/` → `shared/`
- **现状缺少**：
  - `widgets/` — 无跨页面可复用业务组件层
  - `entities/` — 无领域层（model / api / ui 分离）
  - `shared/config/` — 无环境变量配置层
  - `app/providers.tsx` — Provider 组合写在 `App.tsx` 里而非独立文件
- **影响**：低 — 当前项目功能单一，不强需中间层

### 7. `build` 脚本缺少类型检查

- **文档要求**：`"build": "tsc && vite build"` — 构建前先做类型检查
- **现状**：`"build": "vite build"` — 跳过了 `tsc` 类型检查
- **影响**：中 — 可能让类型错误的代码被打包

### 8. 代码分割 / 懒加载缺失

- **文档要求**：组件懒加载覆盖率 > 80%，首屏 JS < 200KB（Gzip）
- **现状**：无任何 `React.lazy` / 动态 `import()`，整个应用打包为单 chunk：
  - `index.js` 903KB（Gzip 284KB） — **远超 200KB 基线**
- **影响**：高 — 首屏性能

### 9. Conventional Commits 规范

- **文档要求**：提交信息遵循 `feat:`, `fix:`, `refactor:` 规范
- **现状**：项目尚未初始化 Git 仓库，无任何提交规范约束
- **影响**：低 — Git 初始化时一并配置即可

### 10. Tailwind 配置方式差异（已合理）

- **文档提到**：`tailwind.config.ts` 主题定制
- **现状**：使用 Tailwind CSS 4.x CSS-first 配置（在 `globals.css` 中用 `@theme`），无 `tailwind.config.ts`
- **影响**：无 — 这是 v4 的正确做法，文档描述的是 v3 方式

## 三、优先级汇总

| 优先级 | 差异项 | 建议 |
|--------|--------|------|
| 🔴 高 | 代码分割缺失（284KB Gzip） | Tiptap 等重依赖用 `React.lazy` 懒加载，配置 `manualChunks` |
| 🔴 高 | Shadcn UI 未正式引入 | 引入 Radix UI 基础组件替换手写 Dialog/ContextMenu |
| 🟡 中 | `biome.json` 配置缺失 | 创建配置文件，定义项目规则 |
| 🟡 中 | `build` 脚本缺 tsc | 改为 `"build": "tsc --noEmit && vite build"` |
| 🟡 中 | React Router 未使用 | 视后续扩展需求决定，当前可暂缓 |
| 🟢 低 | FSD 中间层 widgets/entities | 功能扩展时按需添加 |
| 🟢 低 | Husky / Git 规范 | Git 初始化时一并配置 |
| 🟢 低 | React Hook Form | 表单复杂度增加时引入 |
