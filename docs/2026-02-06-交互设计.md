# NanoMind 交互设计文档

> 详细描述所有用户交互流程、状态机、边界情况处理。供从零重构时作为交互参考。

---

## 一、核心交互流程

### 1.1 应用启动流程

```
页面加载
  ├─ 从 localStorage 读取主题设置（默认 Theme A）
  ├─ 检查 URL 参数 ?previewTheme=
  │   ├─ 有值 → 进入主题预览页（不显示常规界面）
  │   └─ 无值 → 进入常规界面
  ├─ 应用主题到 <html> 元素的 data-theme 属性
  ├─ 调用 API 获取文件树
  └─ 渲染空状态（"从左侧选择文档"）
```

### 1.2 打开文件流程

```
用户点击文件树中的文件
  ├─ 检查是否有未保存的变更
  │   ├─ 有 → 弹出浏览器原生 confirm（"有未保存的更改，是否放弃？"）
  │   │   ├─ 取消 → 中止操作，保留当前状态
  │   │   └─ 确认 → 继续执行
  │   └─ 无 → 继续执行
  ├─ 调用 API 读取文件内容（Markdown）
  ├─ 将 Markdown 转换为 HTML
  ├─ 设置编辑器内容
  ├─ 记录原始内容（用于变更检测）
  ├─ 设置当前路径
  ├─ 退出编辑模式
  ├─ 重置变更标记
  ├─ 添加到最近文件列表
  └─ 从内容中提取 TOC 标题
```

### 1.3 编辑流程

```
进入编辑模式
  ├─ 条件：已打开文件 + 当前为阅读模式
  ├─ 触发方式：Ctrl/Cmd+E 或 点击编辑按钮
  └─ 结果：
      ├─ 编辑器变为可写
      ├─ 显示工具栏
      ├─ 顶部栏按钮变为：取消 + 保存

编辑中
  ├─ 每次内容变化时对比原始内容，更新变更标记
  └─ 变更标记控制保存按钮的启用/禁用

保存
  ├─ 触发方式：Ctrl/Cmd+S 或 点击保存按钮
  ├─ 条件：编辑模式 + 有变更 + 非保存中
  ├─ 流程：
  │   ├─ 设置保存中状态（按钮显示加载动画）
  │   ├─ 将 HTML 转换为 Markdown
  │   ├─ 调用 API 保存
  │   ├─ 成功：更新原始内容、重置变更标记、退出编辑模式
  │   └─ 恢复保存状态
  └─ 失败时保持编辑模式

取消编辑
  ├─ 触发方式：Escape 或 点击取消按钮
  ├─ 流程：
  │   ├─ 弹出自定义确认对话框
  │   │   ├─ 标题："放弃修改？"
  │   │   ├─ 描述："确定要放弃未保存的修改吗？"
  │   │   ├─ 按钮：取消 / 放弃
  │   │   └─ 焦点自动聚焦到"放弃"按钮
  │   ├─ 点击"取消" → 关闭对话框，保持编辑模式
  │   └─ 点击"放弃" → 恢复原始内容、重置变更标记、退出编辑模式
```

### 1.4 搜索流程

```
打开搜索
  ├─ 触发方式：Ctrl/Cmd+K
  └─ 结果：打开搜索弹窗，输入框自动聚焦

搜索弹窗交互
  ├─ 输入搜索词
  │   ├─ 防抖 200ms
  │   ├─ 少于 2 字符 → 清空结果
  │   └─ >= 2 字符 →
  │       ├─ 文件名模式：本地过滤文件索引
  │       └─ 全文模式：本地过滤 + API 搜索 → 合并结果
  │
  ├─ 导航列表 = 最近文件 + 搜索结果
  │   └─ 默认选中第一项
  │
  ├─ 键盘导航
  │   ├─ ↑↓ → 移动选中项（循环）
  │   ├─ Enter → 打开选中项
  │   └─ Ctrl/Cmd+D → 切换搜索模式
  │
  ├─ 点击结果 → 打开文件
  └─ 关闭弹窗：Escape 或 点击遮罩层

打开文件后
  ├─ 关闭搜索弹窗
  ├─ 清空搜索词
  └─ 清空搜索结果
```

### 1.5 文件操作流程

#### 新建（通过底部按钮）

```
点击"新建"按钮
  ├─ 弹出新建对话框
  │   ├─ Step 1：选择类型（文档/目录）+ 选择目标位置
  │   └─ 点击"下一步"
  ├─ 触发父组件的 onCreateFile/onCreateDir 回调
  ├─ 弹出名称输入对话框
  │   ├─ 输入名称
  │   ├─ Enter 或 点击"创建" → 调用 API
  │   └─ 名称为空时"创建"按钮禁用
  ├─ 创建成功
  │   ├─ 刷新文件树
  │   ├─ 关闭对话框
  │   └─ 如果是文件 → 自动打开新文件
```

#### 新建（通过右键菜单）

```
右键点击目录 → "新建文档" / "新建目录"
  ├─ 直接弹出名称输入对话框（父目录为右键的目录）
  ├─ 后续流程同上
```

#### 重命名

```
右键点击文件/目录 → "重命名"
  ├─ 弹出重命名对话框
  │   ├─ 预填当前名称（不含 .md）
  │   ├─ 输入新名称
  │   └─ Enter 或 点击"确定" → 调用 API
  ├─ 成功
  │   ├─ 刷新文件树
  │   └─ 如果重命名的是当前文件 → 更新当前路径
```

#### 删除

```
右键点击文件/目录 → "删除"
  ├─ 浏览器原生 confirm："确定要删除吗？此操作不可恢复。"
  │   ├─ 取消 → 中止
  │   └─ 确认 → 调用 API 删除
  ├─ 成功
  │   ├─ 刷新文件树
  │   └─ 如果删除的是当前文件
  │       ├─ 清空当前路径
  │       ├─ 清空内容
  │       └─ 重置变更标记
```

### 2.2 编辑模式状态机

```
                   Ctrl+E / 编辑按钮
 [阅读模式] ────────────────────────────→ [编辑模式]
     ↑                                        │
     │                                        │
     │  Ctrl+S / 保存按钮（成功）              │  Escape / 取消按钮
     │←──────────────────                      ↓
     │                                   [取消确认对话框]
     │                                        │
     │  确认放弃                               │ 取消
     │←──────────────────                      ↓
                                         [回到编辑模式]
```

### 2.3 搜索弹窗状态机

```
 [关闭] ──── Ctrl+K ────→ [打开]
   ↑                         │
   │                         ├─ 输入搜索词 → 更新结果
   │                         ├─ ↑↓ → 移动选中
   │                         ├─ Enter → 打开文件 → [关闭]
   │                         ├─ Ctrl+D → 切换模式
   │                         └─ Escape → [关闭]
   │                                     │
   └─────────────────────────────────────┘
```

## 三、快捷键优先级

当多个快捷键处理器可能冲突时，按以下优先级处理：

1. **搜索弹窗打开时**：搜索弹窗的快捷键优先
   - Escape → 关闭搜索（不触发取消编辑）
   - ↑↓ → 搜索结果导航（不触发默认行为）
   - Enter → 打开搜索结果（不触发默认行为）
   - Ctrl+D → 切换搜索模式

2. **弹窗打开时**（取消确认/新建/重命名）：不处理 Escape 取消编辑

3. **编辑模式**：
   - Escape → 触发取消编辑流程
   - Ctrl+S → 保存
   - 编辑器内部快捷键正常工作

4. **通用**：
   - Ctrl+K → 打开搜索
   - Ctrl+E → 进入编辑模式

## 四、Escape 键行为规则

Escape 键按以下优先级处理（从高到低）：

| 优先级 | 条件 | 行为 |
|--------|------|------|
| 1 | 搜索弹窗打开 | 关闭搜索弹窗 |
| 2 | 取消确认对话框打开 | 不处理（由对话框自身处理） |
| 3 | 新建弹窗打开 | 不处理（由对话框自身处理） |
| 4 | 重命名弹窗打开 | 不处理（由对话框自身处理） |
| 5 | 编辑模式 | 触发取消编辑（弹出确认对话框） |
| 6 | 其他 | 无操作 |

## 五、文件树交互细节

### 5.1 展开/折叠逻辑

- 初始加载时默认展开第一层所有目录
- 点击目录项切换展开/折叠状态
- 展开状态存储在组件内存中（不持久化）
- 文件树数据更新时重新计算第一层展开状态

### 5.2 选中高亮

- 当前打开的文件在文件树中高亮显示
- 高亮样式：主题色半透明背景 + 主题色文字 + 加粗

### 5.3 文件名显示

- 文件显示 `.md` 扩展名（如 `会议记录.md`）
- 如果文件的 `name` 字段不含 `.md`，前端自动追加显示
- 目录显示原始名称

## 六、TOC 交互细节

### 6.1 标题提取

- 从 HTML 内容中解析 H1、H2、H3 标签
- 为每个标题生成唯一 ID（slugify 处理：小写、空格转连字符、去除特殊字符）
- 相同文本的标题通过追加数字后缀去重

### 6.2 点击跳转

- 点击 TOC 项后，主内容区平滑滚动到对应标题位置
- 滚动偏移量：标题 offsetTop - 8px
- 如果通过 ID 找不到元素，尝试通过文本内容匹配 H1/H2/H3 标签

### 6.3 滚动高亮

- 监听主内容区的滚动事件
- 遍历所有标题 DOM 元素
- 找到最后一个 `offsetTop - 12 <= scrollTop` 的标题
- 将对应 TOC 项标记为激活状态
- 页面加载后立即执行一次高亮计算

### 6.4 响应式行为

- 屏幕宽度 < xl（1280px）时隐藏 TOC 侧边栏
- 主内容区自动填充空出的空间

## 七、对话框交互规范

### 7.1 通用规范

- 打开时有遮罩层（点击遮罩可关闭）
- 最大宽度 `md`（约 448px），搜索弹窗为 `2xl`（约 672px）
- 按 Escape 可关闭
- 打开时自动聚焦到主要交互元素

### 7.2 各对话框焦点管理

| 对话框 | 打开时聚焦 |
|--------|-----------|
| 搜索弹窗 | 搜索输入框 |
| 新建名称输入 | 名称输入框（autoFocus） |
| 重命名 | 名称输入框（autoFocus） |
| 取消编辑确认 | "放弃"按钮 |

### 7.3 链接输入

- 链接输入是工具栏下拉菜单内的内嵌表单，不是独立对话框
- 包含 URL 输入框 + 确定按钮
- 按 Enter 可快捷提交
- 编辑已有链接时预填当前 URL

## 九、边界情况处理

### 9.1 内容为空

| 场景 | 处理 |
|------|------|
| 文件树为空（无任何文件） | 正常显示空的文件树区域 |
| 文档内容为空 | 编辑器显示占位文字："开始编写..." |
| 搜索无结果 | 搜索结果区域为空，仍显示最近文件 |
| TOC 无标题 | 显示"暂无目录" |

### 9.2 路径处理

| 场景 | 处理 |
|------|------|
| 文件名含中文 | URL 编码传输，后端解码 |
| 文件名含空格 | URL 编码传输，后端解码 |
| 文件名含特殊字符 | URL 编码传输，后端解码 |
| 路径包含 `..` | 后端路径校验拒绝 |

### 9.3 并发操作

| 场景 | 处理 |
|------|------|
| 快速连续点击保存 | 保存中状态防止重复提交 |
| 搜索快速输入 | 防抖处理（200ms） |
| 侧边栏搜索快速输入 | 防抖处理（300ms） |

### 9.4 错误处理

| 场景 | 处理 |
|------|------|
| API 请求失败 | 控制台输出错误，用户无明显提示 |
| 文件不存在 | 返回 null/空，不崩溃 |
| 创建同名文件 | API 返回 409，设置错误状态 |
